# 🖥️ Frontend Angular – Week 1 Assessment

A Week 1 assessment project for a Frontend Angular training program, focused on foundational **HTML** and **CSS** skills. This project demonstrates the ability to structure web pages semantically and style them using modern CSS techniques — the building blocks required before diving into Angular.

---

## 📁 Project Structure

```
Frontend-Angular-Week-1-Assessment/
└── Week-1-Assessment-HTML-CSS/
    ├── index.html        # Main HTML page
    └── styles.css        # Stylesheet
```

---

## 🛠️ Technologies Used

| Technology | Purpose              | Usage  |
|------------|----------------------|--------|
| HTML5      | Page structure & content | 32.7% |
| CSS3       | Styling & layout         | 67.3% |

---

## 📌 Topics Covered

- Semantic HTML5 elements (`<header>`, `<main>`, `<section>`, `<footer>`, etc.)
- CSS selectors, specificity, and the box model
- Flexbox / Grid-based layouts
- Typography and color styling
- Responsive design fundamentals

## 🚀 Getting Started

No installation or build tools required. Simply open the project in a browser:

### Option 1 – Open directly

```bash
# Clone the repository
git clone https://github.com/cser-naivedya-raj/Frontend-Angular-Week-1-Assessment.git

# Navigate into the assessment folder
cd Frontend-Angular-Week-1-Assessment/Week-1-Assessment-HTML-CSS

# Open index.html in your browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

### Option 2 – Live Server (VS Code)

1. Open the folder in **VS Code**
2. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
3. Right-click `index.html` → **Open with Live Server**
